Overview

From reactive dispatching to strategic orchestration

This project presents a hierarchical heuristic reinforcement learning framework for dynamic AGV dispatching in AI server reliability testing workshops. The method combines strategic planning with low-level execution control, enabling multi-load AGVs to respond more effectively to congestion, disturbances, and changing task priorities.

Core Contributions

  • Hierarchical dispatching that separates strategic planning from low-level execution
  • D3QN high-level policy with invalid action masking
  • Opportunistic picking and intent coordination for stronger AGV collaboration
  • Improved cycle time and waiting time under dynamic disturbances
